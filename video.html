Video Element in HTML

The video element embeds a media player which supports video palyback into the documents

The control attribute give the control to the media player. If this attribute is present the browser will offer controls to allow the user to control the audio/video playback, incliuding volume and pause/resume playback.

The autoplay attribute helps to autoplay the video when client using that web-page.

The loop attribute helps to video to continually repeate the video.
